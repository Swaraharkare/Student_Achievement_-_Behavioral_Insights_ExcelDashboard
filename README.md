# Student-Achievement-&-Behavioral-Insights (Interactive Dashboard creation using MS Excel )
An interactive Excel Executive Dashboard that transforms raw educational data into actionable insights. This project analyzes the correlation between demographics, behavioral habits, and academic performance to help administrators identify key drivers of student success.

Dataset Used
-<a href="https://www.kaggle.com/datasets/kundanbedmutha/student-performance-dataset">Student Performance Dataset</a>
